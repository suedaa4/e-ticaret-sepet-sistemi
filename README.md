# 🛒 E-Commerce Cart System (E-Ticaret Sepet Sistemi)

Bu proje, **JavaScript** kullanılarak nesnelerden oluşan bir diziyi (Array of Objects) döngüye sokmayı ve birikimli hesaplama mantığını anlamayı amaçlayan orta seviye bir algoritma alıştırmasıdır.

## 🚀 Özellikler
* Sepete dinamik olarak isim ve fiyat bilgisi içeren ürün nesneleri ekleme (`.push()`).
* Sepetteki tüm nesneleri `.forEach()` döngüsü ile dönerek fiyat toplama algoritması çalıştırma.
* Hesaplanan sepet tutarını `return` mekanizması ile dışarı aktarma.
